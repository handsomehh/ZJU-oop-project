本作业是基于cmd命令行窗口实现的日记本，具体使用方式如下：
在含有四个.exe文件的文件夹下，可以执行这些命令和操作
1、pdadd命令：
在cmd中，输入pdadd [参数1]，参数1是一个文件路径，缺省时默认执行对1.txt的操作，如果输入了参数1，将打开该参数对应文件，请保证这个文件已经存在，否则提示“Can't open file”
打开文件后，在第一行输入日期，推荐格式为年/月/日（如2002/01/03），不过无论如何输入，需要保证这些输入的日期可以比较大小，否则会乱序存放
此外，日期的大小比较是按照字符串直接比较的方式实现，使用为确保正确性请保证输入格式一致（比如2002/12/3会比2002/9/9小，所以最好都是2002/12/03和2002/09/09这种方式输入）
在第二行之后输入正文，直到读取到在单独的一行里只存在“.”时退出。
当这个输入的日期已经存在于目标文件中，将替代原有日期的内容为当前输入的内容
文件中的日记默认为按日期早晚顺序存放，越早的日期放在越前面
2、pdlist命令：
在cmd中，输入方式有三种：
	pdlist 此时默认打开1.txt并遍历输出其中的内容
	pdlist [参数1] 参数1是文件路径，此时默认打开文件路径并遍历输出所有内容
	pdlist [参数1][参数2] 默认打开1.txt，其中参数1 是日期下界，参数2是日期上界，需要保证输入日期格式和之前文件中存在的格式相同，至少也要保证这个日期可以和文件中的日期比较，否则不一定输出正确结果
	pdlist [参数1][参数2][参数3] 参数1是文件路径，参数2、参数3分别是日期上下界，将输出在这个日期区间的内容
3、pdremove命令：
在cmd中，输入方式有两种：
	pdremove [参数1] 参数1是具体日期，将默认打开1.txt并删除对应日期的日记，删除成功返回0，否则返回-1
	pdlremove [参数1][参数2] 参数1是文件路径，参数2是日期，将删除参数1对应文件中的参数2对应日记，返回值同上
4、pdshow命令：
在cmd中，输入方式只有一种，该命令必须配合重定向来使用
如果想要查询对应文件中指定日期内容，需要将文件输出重定向至该程序输入，如（type 1.txt | pdshow [参数1]）参数1 是输入的日期，type命令将输出1.txt的内容，而管道会将内容重定向到pdshow
此外，可以用pdlist配合使用，如 pdlist | pdshow [参数1]，这个命令将输出1.txt的内容中对应日期为参数1的日记项。
无论如何，输入到pdshow的内容需要是和日记文件格式相同的内容，参数1保证查询内容为对应日期，具体的重定向方式可以自由实现，而pdshow中使用cin.eof()判断输入的结束，来确保程序最终输出的正确性
